# Что я сделал и изучил

Этот проект — моя практическая работа с Framer Motion для анимации и интерактивности UI-компонентов в React. В репозитории реализованы различные дизайн-компоненты и паттерны анимаций, каждый из которых можно посмотреть и протестировать в интерактивном режиме. Между компонентами можно переключаться через выпадающий список в правом нижнем углу.

## Вот что я реализовал и изучил:

- Настроил проект на React с помощью Vite и TypeScript для быстрой разработки и типизации.
- Интегрировал TailwindCSS для стилизации интерфейса.
- Изучил и применил Framer Motion и motion для создания сложных анимаций и интерактивных эффектов.
- Реализовал коллекцию дизайн-компонентов, каждый из которых демонстрирует отдельный паттерн анимации или взаимодействия.
- Практиковался в работе с drag & drop, анимациями по скроллу, жестами, переходами между состояниями и др.
- Использовал Zustand для управления состоянием в некоторых компонентах.
- Оптимизировал рабочий процесс для разработки, тестирования и масштабирования анимированных UI-компонентов.

## Описание дизайн-компонентов

- Counter Demo — простой счетчик с анимацией, использует Zustand для управления состоянием.
- Pulsing Button — пульсирующая кнопка с бесконечной анимацией масштаба и цвета.
- Bouncing Loader — анимированный индикатор загрузки с прыгающими точками.
- Flipping Card — карточка с анимацией переворота по клику.
- Animated Card — карточка с анимацией наведения, перетаскиванием и изменением масштаба.
- Stagger Animation — демонстрация последовательной (staggered) анимации элементов.
- Animated Gallery — галерея изображений с анимацией появления и скрытия.
- Range Slider — слайдер с анимированной кнопкой, реагирующей на изменение значения.
- Draggable Box — перетаскиваемый блок с изменением цвета и отображением координат.
- Scroll Animation — анимация элементов при прокрутке страницы.
- Animation Component — анимация изображения и текста, зависящая от скролла.
- Swipeable Card — карточка, реагирующая на свайпы.
- Gesture Image Gallery — галерея изображений с навигацией с помощью жестов (свайпы вверх/вниз).
- Carousel — карусель изображений с анимацией переходов и миниатюрами.
- Sidebar — анимированная боковая панель с фильтрами и плавным появлением/скрытием.
- Scroll Carousel — горизонтальная карусель карточек, управляемая вертикальным скроллом.
- Sticky Nav — закрепленная навигация с анимацией при прокрутке.
- Progress Steps — пошаговый прогресс-бар с анимацией переходов между этапами.
