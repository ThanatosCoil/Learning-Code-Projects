# Что я сделал и изучил

Этот проект — мой практический эксперимент с Style Dictionary для управления и генерации дизайн-токенов в разных форматах.

## Вот что я реализовал и изучил:

- Настроил проект Style Dictionary для централизованного хранения дизайн-токенов (цвета, отступы, типографика) в JSON-файлах.
- Организовал токены по категориям (color, spacing, typography) для удобства и масштабируемости.
- Настроил автоматическую генерацию переменных в форматах CSS, SCSS и JavaScript с помощью Style Dictionary.
- Практиковался в настройке процесса сборки через файл config.json.
- Изучил, как поддерживать единые и переиспользуемые дизайн-токены для разных платформ и технологий.
- Оптимизировал рабочий процесс для поддержки единого источника правды для дизайн-значений в UI-разработке.
